Para quem é este livro ?

Este livro é principalmente para estudantes que estão fazendo um curso sobre a linguagem C++, para aqueles que desejam estudar a linguagem C++ por conta própria e para programadores que têm experiência com C e desejam avançar para C++. Também pode ser útil para instrutores do curso de C++ que procuram exemplos explicativos de programação para adicionar em suas aulas.

Qual é o foco deste livro ?

O foco deste livro é fornecer uma sólida introdução à linguagem C++ e ao conhecimento de programação por meio de um grande número de exemplos práticos e conselhos significativos. Inclui mais de 500 exercícios e exemplos de dificuldade progressiva para ajudar o leitor a entender os princípios C++ e ver como os conceitos podem se materializar no código.

Quais são os exemplos e exercícios ?

Os exemplos são projetados para serem curtos, concretos e substanciais, dando rapidamente ao leitor a capacidade de entender como aplicar correta e eficientemente os recursos da linguagem C++ e obter um sólido conhecimento de programação. Tenha certeza de que, se você for capaz de entender os exemplos deste livro e resolver os exercícios, poderá prosseguir com segurança para editar programas maiores, poderá desenvolver seus próprios aplicativos e certamente terá estabelecido uma sólida base conceitual e prática fundamental para expandir seus conhecimentos e habilidades.
